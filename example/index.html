👉 Basic Chart

Bar chart (vertical)

$chart(dom, data_d1_m1)
  .bar({x: 'field', y: 'value'})
  .render();

Bar chart (horizontal)

$chart(dom, data_d1_m1)
  .bar({x: 'field', y: 'value'})
  .transpose()
  .render();

Grouped Bar Chart (vertical)

$chart(dom, data)
  .bar({
    x: 'field1|field2', 
    y: 'value'
  })
  .render();

Grouped Bar Chart (horizontal)

$chart(dom, data)
  .bar({
    x: 'field1|field2', 
    y: 'value'
  })
  .transpose()
  .render();

👉 interaction analysis

📐 show the highest

快速在图表中找到标记出来最大值

chartInstance.highlight('max', number).repaint()
number 为可选参数，默认1

📐 show the lowest

快速在图表中找到标记出来最小值

chartInstance.highlight('min', number).repaint()
number 为可选参数，默认1

🗂 sort

选择排序方式，并且可以在动效中清晰的看到数据移动关系

chartInstance.sort('descending').repaint()

🧮 Compare with target (kufri)

添加一组目标数组，快速查看哪些数据超过目标值

chartInstance.compare(data, 'target').repaint()

⚖️ Compare with another dataset

添加一组目标数组，快速查看哪些数据超过目标值

chartInstance.compare(data, 'data').repaint()

🔧 Add or delete data set

数据量增加或者去掉

chartInstance.push(data).repaint()
chartInstance.pop(data).repaint()
chartInstance.shift(data).repaint()
chartInstance.unshift(data).repaint()
