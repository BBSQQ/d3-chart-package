👉 Basic Chart

Bar chart 柱形图

[
  { key: 'A', value: 620 },
  { key: 'B', value: 450 },
  { key: 'C', value: 970 },
  { key: 'D', value: 780 }
]

$chart(dom, data)
  .x('key')
  .y('value')
  .bar()
  .render();

Bar chart 条形图

[
  { key: 'A', value: 620 },
  { key: 'B', value: 450 },
  { key: 'C', value: 970 },
  { key: 'D', value: 780 }
]

$chart(dom, data)
  .x('key')
  .y('value')
  .bar()
  .transpose()
  .render();

👉 interaction analysis

📐 show the highest

快速在图表中找到标记出来最大值

$chart()...highlight('max', number).repaint()
number 为可选参数，默认1

📐 show the lowest

快速在图表中找到标记出来最小值

$chart()...highlight('min', number).repaint()
number 为可选参数，默认1

📏 sort

选择排序方式，并且可以在动效中清晰的看到数据移动关系

$chart()...sort('descending').repaint()

📏 Compare with target (kufri)

添加一组目标数组，快速查看哪些数据超过目标值

$chart()...compare(data, 'target').repaint()

📏 Compare with another dataset

添加一组目标数组，快速查看哪些数据超过目标值

$chart()...compare(data, 'data').repaint()

📏 Add or delete data set

数据量增加或者去掉

$chart()...push(data).repaint()
$chart()...pop(data).repaint()
$chart()...shift(data).repaint()
$chart()...unshift(data).repaint()
